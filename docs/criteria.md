# Inclusion Criteria

## In Scope

- Lesson planning and teaching copilots
- Assessment and feedback workflows
- Language learning support
- Tutoring and learner support
- Content creation and curriculum design
- School operations and administration
- Teacher professional development tied to instructional practice
- Real education case studies
- Education-focused open-source projects

## Out of Scope

- AI literacy programs
- General AI learning courses
- Paper-only collections without practical applications
- Generic AI tools with no clear educational workflow

## Curation Layers

### Main List

Use for entries with clear educational value and strong public usefulness.

### Case Studies

Use for real implementations in educational settings.

### Watchlist / Emerging

Use for promising but not yet fully validated entries.
